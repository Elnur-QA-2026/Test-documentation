# Test-documentation
Avtomobil filtrasiya modulunun testləri
Positive tests:
Nəqliyyat kateqoryasına daxil oldum və avtomobil alt kateqoriyasını seçdikdən sonra məhsulları göstərdi
Toyota markasını seçdikdən sonra ancaq satışda olan toyota avtomobilərini göstərdi
Qiymət filtrasyası işləyir
Avtomobilləri bir neçə xüsusiyyətlərə uyğun olaraq filtrasiya etmək mümkündür
Valyutanı (USD, AZN) dəyişmə düyməsi düzgün işləyir
Satışda olan avtomobili yerləşdirən satıcı ilə sayt vasitəsilə əlaqəyə keçmək mümkündür. Sayt sizə bir neçə üsul təqdim edirş
Neqativ tests:
Satışın hansı şərtlə olduğunu filtrasiya etmək mümkün deyil. Yəni kreditdə və ya nağd satışda olmaqını
Ucuzdan bahaya filtrasiya zamanı elanlarda yalnız avtombillər yox, həmçinin ehtiyyat hissələri və kirayə üçün təkliflərdə göstərir. Axtarışa maneə olur.
Filtrasiyada sükan bölməsi olsada heç bir filtrasiya eləmir. Bu istifadəçi səhvidə ola bilər
Müəyyən bir filtrasiya etdikdən sonra həmin bölgüyə uyğun elanlar bitdikdən sonra saytın oz təklif etdiyi məhsullar gəlir. Elanlar arasında keçid gözə çarpmır.
Axtarış yerine marka, model ve ilini yazdıqdan gösterilən avtomobillərin bəziləri axtarışa uyğun deyil. Əsasən ilini səhv göstərir.
